# 打包类库
> 使用webpack 可以很方便生成合并压缩的js文件

## webpack 如何管理前端包

**使用npm包管理**
```shell
    npm install axios --save
    npm install vue --save
```
webpack.config.js
```javascript
module.exports = {
    resolve: {
        alias: {
            'vue$': path.resolve(__dirname, '../node_modules/vue/dist/vue.common.js')
        }
    }
    
}
```
