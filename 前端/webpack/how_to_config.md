# How To Config ?
> 如何去配置 webpack.config.js

## DEMO
```javascript
{
  context: __dirname + "/app",
  entry: "./entry",
  output: {
      path: __dirname + "/dist",
      filename: "bundle.js"
  }
}

```


## 配置项

| name | description | type | default |
| -- | -- | -- | -- |
| `context` | 1:2 | 2:2 | 3:2 |
| `context` | 1:2 | 2:2 | 3:2 |
